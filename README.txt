1. Plane Simulation using OpenGL
2. Visual Studio 2012 is used to deploy
3. 9 IMAGES are used for texture which can be found in
   image folder inside ConsoleApplication1
4. Features: 
	a) Plane Landing both manual and auto technology
	b) Plane Crash in Manual Landing
	c) Plane Take-off
	d) Plane Crash during Take-off
5. Run "ConsoleApplication1.sln" and command line output
   statement is provided to help to drive the aeroplane
6. "VIDEO.mp4" is illustrating the whole application
